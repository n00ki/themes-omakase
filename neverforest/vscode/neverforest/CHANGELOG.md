# Change Log

## [0.0.3] - 2026-02-05

### Changed
- Improved highlight colors for better visibility and theme consistency
- Modified indicators now use aqua (#7fbbb3) - the theme's signature color
- Created/added indicators now use gold (#dbbc7f) for warm accent on new items
- Info colors updated to aqua (#7fbbb3) for palette consistency

## [0.0.2] - 2026-01-09

### Changed
- Aligned UI colors for consistency
- Properties now use teal (#83c092)
- Darkened title bar, inactive tabs, and panel backgrounds (#0c1116)
- Improved list/menu selection visibility (#1b2733)
- Added transparency to line numbers and active line highlight
- Fixed import/export variable colors to match standard variables

## [0.0.1] - 2025-11-11

### Added
- Initial release of Neverforest theme for VSCode
